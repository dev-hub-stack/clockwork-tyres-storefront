What is Clockwork Tyres?

Clockwork Tyres is the new tyre-first evolution of the Clockwork B2B storefront. Retailers can search by vehicle or by size, see their own stock first, and surface approved supplier stock inside the same catalog journey with supplier identity hidden.

Features

Search

Search by vehicle.

  • Start with make, model, year, and variant to move quickly into the correct tyre catalog.

Search by size.

  • Use width, aspect ratio, rim size, and service details to narrow down the exact tyre fit.

Search-led storefront.

  • The launch keeps the same search-first Clockwork rhythm instead of turning the storefront into a generic landing page.
Features

Inventory

Own stock first.

  • Retailers see their own stock first in the storefront, with low quantities visible when stock is limited.

Merged supplier stock.

  • Approved supplier stock appears behind the same tyre entry, with supplier identity hidden from the retail storefront.
Platform

Supplier Portal

Wholesale-ready.

  • Suppliers manage tyres, pricing levels, inventory, warehouses, and procurement approvals from the CRM side.

Retail preview mode.

  • Supplier preview reuses the same storefront shell, but keeps cart and checkout disabled.

Future category seam.

  • Tyres launch first, while the platform structure stays open for wheels and more categories later.

Start with the Clockwork storefront feel users already know, then layer in the new tyre catalog, supplier stock, and admin platform behind it.
